会社概要

当社は、日本企業向けに越境EC・マルチテナント型コマースプラットフォームを提供するテクノロジーカンパニーです。 Next.js / React / TypeScript を中心としたモダンスタックと、Tailwind CSS・shadcn/ui を組み合わせることで、 シンプルで安心感のある管理画面を実現しています。

  • Reliable：日本語による丁寧なサポート

サービス概要

3つのモジュールを組み合わせることで、貴社のフェーズに合わせた段階的な導入が可能です。 MVP から本番運用までを一気通貫で支援します。

モジュール 1

企業公式サイト構築

/, /company, /services, /cases, /contact など、企業サイトに必要なページを整備します。

  • Next.js + Tailwind ベース
  • 多言語対応（日本語・英語・中国語）
  • SEO最適化（Metadata API）
  • SSG / ISR による高速表示

モジュール 2

販売者ダッシュボード

/auth/login, /dashboard, /dashboard/products など、販売者向け管理画面を提供します。

  • メール＋パスワード認証（next-auth）
  • 商品管理・注文管理・財務ダッシュボード
  • AIアシスタントの統合（翻訳・商品提案）

モジュール 3

商品公開サイト

/shop/[shopId], /shop/[shopId]/product/[productId] 形式でマルチテナント構成を実現します。

  • 多言語の商品名・説明文の表示
  • 価格・在庫・画像の表示
  • MVPでは「閲覧のみ」（決済は後続フェーズ）
